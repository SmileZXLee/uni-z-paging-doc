### 全屏Loading配置

| 参数                                     | 说明                             | 类型    | 默认值 | 可选值 |
| :--------------------------------------- | :------------------------------- | :------ | :----- | :----- |
| auto-hide-loading-after-first-loaded     | 第一次加载后自动隐藏loading slot | Boolean | true   | false  |
| loading-full-fixed <Badge text="2.0.9"/> | loading slot是否铺满屏幕并固定   | Boolean | false  | true   |
